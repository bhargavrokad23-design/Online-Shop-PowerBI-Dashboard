# Online Shop Sales Dashboard (Power BI)

![Dashboard](dashboard.png)

## Project Overview
This Power BI dashboard analyzes online shop sales performance,
profit trends, and customer purchasing behavior.

## Key Insights
• Total Sales: 267K  
• Total Profit: 19K  
• Total Quantity Sold: 3263  
• Highest Sales State: Madhya Pradesh  
• Top Category by Quantity: Clothing

## Tools Used
• Power BI  
• Excel  
• Data Visualization

## Dataset
The dataset contains information about:
• Sales amount
• Profit
• Quantity
• State
• Customer name
• Product category
